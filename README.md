# Stockify 📦

Stockify is a modern, fast, and intuitive inventory management system designed for small businesses. It provides a simple yet powerful interface to track stock easily and boost productivity, built with a premium SaaS-like aesthetic.

![Stockify Banner](https://stockify-project-tau.vercel.app/images/stockify-social-banner.png)

## ✨ Features

### 🛠️ Core Functionality (CRUD)
* **Add Products:** Easily add new products to your inventory with a user-friendly form including Name, Price, Category, Description, and an Image.
* **View Inventory:** Display all products in a beautiful, responsive grid layout with custom-designed premium cards.
* **Update Products:** Quickly edit and update product details without reloading the page.
* **Delete Products:** Remove individual products or use the **Clear All** feature to wipe the entire inventory at once.

### 🧠 Smart Data Management
* **Local Storage Persistence:** All inventory data is seamlessly saved to the browser's `localStorage`. Your data is never lost, even if you refresh or close the tab.
* **Real-Time Search & Filtering:** Instantly search your inventory by Product Name or Category using a dynamic search bar.

### 🛡️ Robust Validation & Feedback
* **Advanced Form Validation:** Real-time Regular Expression (Regex) validation for all fields ensures clean and structured data entry.
* **Visual Validation Feedback:** Instant inline error messages and color-coded input borders (green for valid, red for invalid).
* **SweetAlert2 Integration:** Native browser alerts are completely replaced with beautiful, animated popup modals for confirmations, success messages, and errors.

### 🎨 Premium UI / UX Design
* **Modern SaaS Aesthetic:** Clean typography, soft drop-shadows, glassmorphism UI elements, and smooth micro-interactions.
* **Fully Responsive:** Built with Bootstrap 5, the layout perfectly adapts to desktops, tablets, and mobile devices.
* **Elegant Empty States:** Displays a professional, user-friendly empty state graphic when no products are in the inventory.
* **Image Handling:** Utilizes `object-fit: cover` to ensure product images are always beautifully proportioned, regardless of their original dimensions.

### 🚀 SEO & Meta Optimization
* **Social Media Ready:** Fully configured Open Graph (OG) and Twitter Card tags to display a custom banner and description when the link is shared.
* **Search Engine Optimized:** Structured title tags, keywords, and meta descriptions designed to improve search ranking visibility.

## 💻 Tech Stack
* **HTML5:** Semantic and accessible markup.
* **CSS3:** Custom styles, Flexbox/Grid, transitions, and animations.
* **Bootstrap 5:** Responsive grid system and utility classes.
* **Vanilla JavaScript (ES6+):** Complete application logic, DOM manipulation, and local storage management (No JS frameworks used).
* **FontAwesome 6:** Premium icons for buttons, ratings, and UI elements.
* **SweetAlert2:** Interactive and stylish popup alerts.
